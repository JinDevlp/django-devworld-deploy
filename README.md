## Functionality
- Django Web Application
- Django REST API
- CRUD
- User Authorization
- Email upon Registering
- Message with other Users
- Keep data safe using Amazon RDS(PostgreSQL) and S3 bucket
- Serverless Application

## What is it?
- Website Community for developers to connect
- Make posts on topics such as Jobs, Projects, Questions, etc
- Used Zappa along with setting up my own S3 Bucket and Amazon RDS

## What did I learn?
- CRUD functionality
- Django REST Framework
- API
- Migrate and store User uploaded content to S3 bucket
- Store User's data safely in Amazon RDS(PostgreSQL) and access it locally
